# juejue-h5

移动端记账本设计

涉及技术：
- react hooks
- react-router-dom
- zarm 移动端 UI 框架(对 react18 适配不好)
- 移动端适配
  - lib-flexible
  - rem

实现功能：
- 用户登录或注册
- 账单分类列表展示
- 账单数据可视化
- 个人中心
- 都可进行实时编辑修改

接口：http://api.chennick.wang 或见 [后端代码实现](https://github.com/Jinx-FX/juejue-server)
